Conociendo la estructura básica de HTML

HTML (HyperText Markup Language) es el lenguaje principal utilizado para estructurar contenido en la web. Conocer la organización de sus etiquetas es fundamental para crear sitios bien organizados y accesibles.

En este documento veremos el uso correcto de etiquetas como <header>, <nav>, <section>, <article>, <aside> y <footer>.

Barra de navegación

La etiqueta <nav> define un conjunto de enlaces de navegación que ayudan al usuario a recorrer las distintas secciones del sitio web. Es importante para la usabilidad y también para los motores de búsqueda que indexan el sitio.

Sección principal

El elemento <section> se usa para dividir el contenido en bloques temáticos. Cada sección puede tener su propio encabezado y se recomienda cuando el contenido sigue una estructura lógica o narrativa.

Por ejemplo, una página sobre historia puede tener secciones por siglos, y una tienda online puede tener secciones para cada categoría de productos.

¿Qué es un artículo en HTML?

La etiqueta <article> define un contenido que tiene sentido por sí mismo y que podría ser redistribuido de manera independiente. Es común su uso en blogs, noticias, comentarios y redes sociales.

Este elemento es ideal cuando el contenido tiene un título, cuerpo y hasta autor propio. Se puede anidar dentro de secciones y también puede contener otras etiquetas como listas, imágenes o citas.

Ejemplo de uso:

<article>
  <h2>Cómo cuidar tus plantas en verano</h2>
  <p>El calor puede afectar a tus plantas, pero con estos consejos lograrás mantenerlas saludables...</p>
</article>

Esto permite crear contenido bien organizado y fácil de compartir.

Contenido complementario

El elemento <aside> contiene información secundaria o de apoyo. Por lo general se sitúa al costado del contenido principal, pero también puede aparecer dentro de artículos.

Se usa para colocar listas de enlaces relacionados, glosarios, definiciones, banners o notas aclaratorias. Es útil para ofrecer contexto sin interrumpir el hilo principal del contenido.

Pie de página

El <footer> suele ubicarse al final del documento o de una sección. Contiene información como el autor, enlaces legales, contacto o fecha de publicación.

También se puede usar dentro de artículos o secciones específicas, no solo al final de la página completa.
